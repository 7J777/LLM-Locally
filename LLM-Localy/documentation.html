How to use the APP

Avant de commencer, à utiliser l'application, assurez-vous d'avoir installé les deux containers Docker suivants : "Open-webui" et "Ollama". Vous pouvez récupérer l'image d'Ollama en tapant dans votre terminal : 


docker run -d -v ollama:/root/.ollama -p 11434:11434 --name ollama ollama/ollama --restart always

Pour Open-webui, vous pouvez récupérer l'image en tapant dans votre terminal : 

docker run -d -p 3000:8080 -e OLLAMA_BASE_URL=http://host.docker.internal:11434 -v open-webui:/app/backend/data --name open-webui --restart always ghcr.io/open-webui/open-webui:main

Une fois que vous avez installé les deux containers, vous pouvez verifier qu'ils sont bien actif via des outils comme Docker Desktop ou en tapant dans votre terminal : 

docker ps
Si tu utilse Docker Desktop tu devrais voir deux container actifs, un nommé "ollama" et un nommé "open-webui"; si ce n'est pas le cas appuie sur le boutton "play" pour les démarrer. Si tu utilise la ligne de commande, tu devrais voir deux lignes avec les noms "ollama" et "open-webui" en fin de ligne.

Si tout est bon, tu peux ouvrir l'application sur ton ordinateur.
